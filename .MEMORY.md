## 项目上下文

### 日程表 (schedule/)
- 后端: FastAPI + SQLite，端口 3000，含 reminders 系统和后台 worker 线程
- 前端: React + Vite + Tailwind，端口 5173/5174
- 数据模型: date(YYYY-MM-DD) + priority(int) + completed(bool) + note
- 提醒: 支持 once/daily/weekly/monthly，可配 webhook
- 监听器: `/tmp/reminder_monitor.py` 每 10s 轮询 DB 跟踪提醒触发
